Class: Client

Client(info, netInfoUrl, netTokenUrl)

Represents a Node within a browser environment.

The Client:

  • Retrieves router information via HTTP (netInfoUrl).
  • Connects to routers via browser WebSocket.
  • Retrieves an authentication token via netTokenUrl, using credentials from cookies.
  • Cannot accept incoming HTTP or WebSocket connections.

Implements the following 4 abstract methods from the base Node class:

  • updateToken()
  • doFetch(url, options)
  • requestSocket(routerId)
  • closeSocket(socket)
